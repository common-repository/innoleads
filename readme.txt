== Description ==

The InnoLeads plugin allows you to easily connect your wordpress site to InnoLeads.

== Installation ==

1. Create an account at http://customer.strateqleads.com
2. Validate your email address.
3. Install this plugin.
4. Log in to your account in the plugin admin panel.
5. Choose your website.
6. Your active campaigns will now show up on your website.
